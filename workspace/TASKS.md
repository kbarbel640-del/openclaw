## 如何使用

**杜甫**：直接在這裡加任務，寫清楚「完成標準」，設定「模式」欄位
**無極**：
1. 每次覺知循環檢查這個檔案
2. 有高優先任務 → Ralph Loop 模式持續迭代
3. 沒有 → 從探索型任務挑一個
4. 完成後移到「已完成」區塊
5. 發現新任務 → 加到對應優先級

**模式欄位**：
- `運行` — Clawdbot 覺知循環處理（預設）
- `開發` — dev_loop.py 用 Claude Code 在獨立分支上處理
- `人工` — 等杜甫手動處理

**完成標準很重要！** 要能客觀判斷「做完了沒」。
**交付物很重要！** 每個任務必須有具體產出（檔案/清單/修復），不能只寫「已分析」。
