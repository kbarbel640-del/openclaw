# FLClash Skill

控制 FLClash VPN 代理（用於連接 24Bet 資源）。

## 功能

- 啟動/停止 FLClash 代理
- 切換出站模式（規則/全局/直連）
- 截圖查看狀態

## 使用

```bash
# 啟動 FLClash（開啟系統代理 + 規則模式 + 播放）
./start.sh

# 停止 FLClash
./stop.sh

# 查看狀態截圖
./status.sh
```

## 注意事項

- FLClash 和 ZeroTier 是分開的，不要同時使用
- FLClash 用於 24Bet 資源
- ZeroTier 用於 BG666 資源

## 依賴

- peekaboo（macOS UI 自動化）
- 需要輔助使用權限給 node
