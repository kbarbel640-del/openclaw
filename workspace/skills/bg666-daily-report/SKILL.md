# BG666 Daily Report Skill

為 BG666 (Two) 生成工作日報並發送到打卡日報群。

## 核心原則

- **低調務實** — 不自嗨、不堆 emoji、不炫耀
- **考慮他人感受** — 中規中矩，讓同事看得舒服
- **只列事實** — 做了什麼就寫什麼

## 日報格式

```
M/DD 工作日報

1. 項目一
2. 項目二
3. 項目三（如有子項）：
   • 子項 a
   • 子項 b
4. 項目四

打卡下班
```

## 流程

### 1. 收集素材

讀取今日相關對話：
```bash
# 策劃群
python scripts/read_chat.py -5000326699 --limit 50

# 數據日報群  
python scripts/read_chat.py -5173465395 --limit 50

# Albert 私聊
python scripts/read_chat.py 5308534717 --limit 30

# Pilipina 私聊
python scripts/read_chat.py <pilipina_id> --limit 30
```

### 2. 整理內容

從對話中提取今日完成的工作：
- 數據需求交付
- 老闆提問回覆
- 口徑澄清
- 系統開發進度
- 面試相關

### 3. 撰寫日報

**✅ 要**
- 簡潔列點
- 具體但不囉嗦
- 重要的放前面

**❌ 不要**
- emoji 堆砌
- 「✅ 完成」「🎉 搞定」
- 技術細節展開
- 自我表揚

### 4. 發送

```bash
python scripts/send_message.py -5173465395 "日報內容"
```
