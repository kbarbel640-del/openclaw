# Surprise Box 驚喜盒

## 用途
收到群組消息時，從素材庫抽一條作為招呼或加料，讓每次互動都有小驚喜。

## 使用方式

1. 讀取 `references/pool.json`
2. 根據情境選分類（或隨機）
3. 抽一條未用過的素材
4. 標記已用（更新 `references/used.json`）

## 分類

| 分類 | 說明 | 適用場景 |
|------|------|----------|
| quotes | 中文金句 | 任何時候 |
| english | 英文單字/片語 | 學習時段 |
| trivia | 冷知識 | 閒聊 |
| greetings | 創意招呼 | 秒回第一句 |
| weather_puns | 天氣梗 | 天氣相關話題 |

## 補貨規則
- 素材用到剩 20% 時提醒補貨
- 補貨時一次加 50+ 條
- 不需要每日自動補，手動觸發即可

## 多訊息連發策略
1️⃣ 秒回：從 greetings 抽一條（不用思考）
2️⃣ 回應內容：正常回覆對方說的話
3️⃣ 加料：從 quotes/english/trivia 抽一條
