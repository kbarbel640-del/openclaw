# PTCG 規則速查（給無極用，教 Ryan 對戰）

## 🎯 勝利條件（三選一）
1. **拿完 6 張獎勵卡** — 每打倒對手一隻寶可夢，拿一張自己的獎勵卡
2. **打倒對手所有寶可夢** — 對手場上沒寶可夢可替換
3. **對手牌庫抽完** — 回合開始必須抽牌，抽不到就輸

## 🃏 三種卡牌
| 類型 | 功能 |
|------|------|
| **寶可夢卡** | 戰鬥主力，有 HP、招式、屬性 |
| **能量卡** | 燃料，貼在寶可夢上才能放招 |
| **訓練家卡** | 道具/支持者/場地，輔助效果 |

## 🔄 回合流程
1. **抽牌**（必做，抽 1 張）
2. **主階段**（自由行動）：
   - 放基礎寶可夢上替補席（最多 5 隻）
   - 進化寶可夢（不能剛上場就進化）
   - **貼 1 張能量**（每回合限 1 次）
   - 打道具卡（無限次）
   - 打支持者卡（每回合限 1 次）
   - 使用特性（看卡牌說明）
   - 撤退（棄掉撤退費用的能量，換替補席的上場）
3. **攻擊**（回合最後一步，攻擊後結束回合）

## 🏁 開局設置
1. 猜拳決定先後
2. 各抽 7 張手牌
3. 放 1 隻基礎寶可夢到戰鬥區（面朝下）
4. 可放最多 5 隻到替補席（面朝下）
5. 各放 6 張獎勵卡（面朝下，不能看）
6. 翻開，開始！

**沒基礎寶可夢？** → 重抽（mulligan），對手可多抽 1 張

## ⚡ 先手限制（重要平衡機制）
- 先手**不能攻擊**
- 先手**不能打支持者卡**

## 💥 傷害計算
1. 招式基礎傷害
2. 檢查弱點（通常 ×2）
3. 檢查抵抗（通常 -30）
4. HP 歸零 → 被打倒 → 去棄牌堆 → 對手拿 1 張獎勵卡

## 🏷️ 屬性相剋（簡化版）
| 攻擊屬性 | 剋 | 被剋 |
|----------|-----|------|
| 🔥 火 | 🌿 草 | 💧 水 |
| 💧 水 | 🔥 火 | ⚡ 電 |
| 🌿 草 | 💧 水 | 🔥 火 |
| ⚡ 電 | 💧 水 | 🟤 格鬥 |
| 🟤 格鬥 | ⚡ 電/⬛ 暗 | 🔮 超能 |
| 🔮 超能 | 🟤 格鬥 | ⬛ 暗 |

## 🎮 簡化對戰規則（Discord 文字版，3v3）

### 與正式規則的差異
- **3v3** 不是 6v6（更快完成）
- **獎勵卡 3 張**不是 6 張
- **不用能量卡**（簡化！招式直接可用）
- **不用訓練家卡**（簡化！純寶可夢對決）
- **撤退免費**（每回合可換一次）

### 簡化回合
1. 選行動：**攻擊** 或 **換寶可夢**
2. 結算傷害
3. HP 歸零 → 淘汰 → 對手拿獎勵卡
4. 3 張獎勵卡拿完 = 贏

這樣 8 歲小朋友也能在 Discord 打文字對戰！

## 📚 教學順序建議
1. 先玩簡化版（3v3 無能量）→ 建立信心
2. 加入能量機制 → 學策略
3. 加入訓練家卡 → 學組合技
4. 完整 6v6 → 準備去店裡實戰
