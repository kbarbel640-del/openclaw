# 任務交接模板

> 放置路徑：`shared/handoff/{來源}→{目標}/YYYY-MM-DD-{任務名}.md`

---

## 📋 交接單

**來源**：{emoji} {名字}（{元素}）
**目標**：{emoji} {名字}（{元素}）
**日期**：YYYY-MM-DD
**優先級**：🔴 高 / 🟡 中 / 🟢 低

---

### 任務摘要
> 一句話說清楚要做什麼

### 輸入資料
- 檔案路徑或內容摘要
- 相關背景連結

### 期望產出
- 具體交付物描述
- 格式要求（如有）

### 截止時間
- 無 / YYYY-MM-DD HH:MM

### 備註
- 特殊注意事項
- 相關上下文

---

## 📬 回覆格式

完成後在原檔案底部追加：

```
---

## ✅ 交付回覆

**處理人**：{emoji} {名字}
**完成時間**：YYYY-MM-DD HH:MM
**產出**：
- 結果摘要
- 產出檔案路徑

**備註**：
- 額外發現或建議
```

---

## 🔔 通知規則

1. 放檔案後在 Discord 發一句：`{目標emoji} {目標名字}，新任務已放置：{任務摘要}`
2. 完成後在 Discord 回一句：`{來源emoji} {來源名字}，任務完成：{任務摘要}`
3. 不要在 Discord 貼完整內容 — 走檔案省 token
