# 出金 — ATM 無卡提款

## 流程
1. 會員問「X 分數可以洗多少台幣」
2. 計算：分數 ÷ 1.4 = 台幣金額
3. 說明「目前因銀行限額，回分僅能無卡提款」
4. 告知「須至台新銀行 ATM 無卡提款」
5. 會員到 ATM 後：
   - 點「無卡交易」
   - 客服提供截圖（QR code 或提款序號）
   - 會員掃截圖或輸入序號 + 密碼
   - 完成提款
6. 無卡提款僅能以「千」為單位

## 匯率
- 出金：分數 ÷ 1.4 = 台幣
- 例：3900 ÷ 1.4 = 2,785 台幣

## VIP 限制
- VIP2：無法贈禮（VIP3 才開啟）
- 如果 VIP2 且分數不足千的整數倍，金額會被截斷
- 例：VIP2 → 2000×1.4=2,800 分數上限

## 常見問題
- 「我只有郵局帳戶」→ 不需要卡，是無卡交易
- 「怎麼才能贈禮」→ VIP3 才開啟贈禮功能
- 「沒別的辦法嗎」→ 官方規定，無法變通，語氣要友善

## 話術亮點
- 用截圖佐證 VIP 規則（不是口說無憑）
- 拒絕時留後路：「需要服務隨時跟我說」
