# 幣塔客服知識庫

## 目的
累積客服對話經驗，最終目標是讓 AI 能直接代替員工服務客戶。

## 目錄結構
- `sop/` — 標準流程（入金、出金、活動、異常）
- `scripts/` — 話術模板（按場景分類）
- `faq/` — 常見問題與標準回覆
- `cases/` — 典型案例（成功/失敗/邊界情況）
- `customer-types/` — 客戶類型與應對策略

## 數據來源
- 幣塔AI工作回報(兔) — 兔兔的客服對話
- 幣塔AI工作回報(俊) — 小峻的客服對話
- 幣塔AI工作回報(QQ) — QQ的客服對話
- 其他員工群組

## 累積方式
每次處理客服截圖校準報告時，同步提取知識點存入對應分類。
