## 話術原則

1. **快**：展示速度優勢，每次「查詢」都秒回
2. **懂行**：用博弈行業術語（VIP、存款、流水、召回）
3. **算賬**：幫客戶算 ROI，不只說功能
4. **降低門檻**：強調「試用」「不滿意不收錢」
5. **收線索**：每次對話目標 = 拿到微信
