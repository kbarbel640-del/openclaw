# 🏦 Ryan 成長銀行 — Discord Bot

Ryan 的寶可夢主題成長系統！發訊息得 XP、每日打卡、猜卡遊戲、等級稱號。

## 功能

| 功能 | 說明 |
|------|------|
| ⚡ XP 系統 | 發訊息自動得 XP（冷卻 60 秒、每日上限 20） |
| 📊 等級系統 | Lv.1-50，寶可夢主題稱號（公式：50 × Level²） |
| 📅 每日打卡 | `!daily` 打卡 +15 XP，連續打卡有額外獎勵 |
| 🌟 每日寶可夢 | `!pokemon` 每天認識一隻第一世代寶可夢 |
| 🔮 猜卡遊戲 | `!猜卡` 三個提示猜寶可夢，越早猜對 XP 越高 |
| 🏆 排行榜 | `!leaderboard` 查看訓練師排名 |

## 指令表

| 指令 | 說明 |
|------|------|
| `!help` | 顯示所有指令 |
| `!rank` | 查看你的等級、XP、進度條 |
| `!xp` | 查看今日 XP 獲得明細 |
| `!daily` | 每日打卡（別名：`!打卡`、`!簽到`） |
| `!pokemon` | 今日寶可夢知識卡（別名：`!寶可夢`） |
| `!猜卡` | 開始猜寶可夢遊戲（別名：`!guess`、`!quiz`） |
| `!提示` | 猜卡遊戲中看下一個提示 |
| `!放棄` | 放棄當前猜卡遊戲 |
| `!leaderboard` | 排行榜（別名：`!lb`、`!排行`） |

## 安裝 & 執行

```bash
cd bots/ryan-bank
pip install -r requirements.txt
export DISCORD_BOT_TOKEN=你的token
python main.py
```
