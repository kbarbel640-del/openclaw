## 範例 1：標準儲值（銀行轉帳）

**情境**：會員買500，銀行轉帳，匯率130

```
📋 校準報告

對話摘要：
• 會員：tzff265
• 類型：儲值入金（銀行轉帳）
• 金額：NT$ 500
• 匯率：500 × 132 = 66,000
• 方式：銀行轉帳（元大銀行 806）

流程步驟：
1. ✅ 客官報「儲值500」
2. ✅ 回覆收款帳號（元大銀行 20952000788898）+ 風控提醒
3. ⚠️ 會員問「有抽獎嗎／抽獎券」
4. ✅ 追問「哥您說哪一個活動呢」
5. ✅ 收到轉帳成功截圖
6. ✅ 回覆「好的 請稍候」→「以儲禮請同意」
7. ✅ 儲禮完成截圖（tzff265 / 66,000）
8. ✅ 結尾話術「已儲禮完成，祝您爆大分贏大獎～」
9. ✅ 會員再問抽獎券 → 請會員截圖活動內容以便查詢

✅ 全流程完整！

評分：
• 抽獎券問題處理得當 — 沒有亂回答，而是請會員提供截圖確認 👍
```

## 範例 2：超商代碼儲值（含逾期重開）

**情境**：會員超商金爸爸500，忘帶錢包，逾期後重開代碼

```
📋 校準報告

對話摘要：
• 會員：nuib275
• 類型：儲值入金（超商代碼）
• 金額：NT$ 500（含40元手續費）
• 代碼：FUN26 029 317 535 → 逾期重開 FUN26 029 317 922
• 入帳幣量：6,500,000
• 匯率：500 × 13,000 = 6,500,000 ✅

亮點：
• 耐心度：⭐⭐⭐⭐⭐ 會員反覆（忘帶錢包→改天→又要開），全程溫柔不急躁
• 逾期處理：主動說明「繳費期限2小時，逾期跟我說，我再幫您開立」
• 加持話術：「祝您把把開免遊」→ 很好的客製化祝福
```

## 範例 3：黑名單查詢（敏感資訊保護）

**情境**：會員代朋友問為什麼被黑名單

```
📋 校準報告

對話摘要：
• 會員：（未具名，幫朋友詢問）
• 朋友LINE名稱：維吉尼亞蘭香
• 類型：黑名單查詢
• 結果：朋友因內部審核未通過，無法交易

關鍵評分：
• 敏感資訊保護：沒有透露審核細節 ✅ 重要！
• 面對追問：堅守底線，只重申規則不透露原因 ✅
• 會員說「我不會直接跟他說」→ 套話技巧，客服正確處理沒有上當 👍

建議補充話術：
「哥不好意思，審核細節屬於內部機制無法對外說明，還請見諒🙏」
```

## 範例 4：出金額度不足

**情境**：會員要回4,320,000分（約3萬台幣），但ATM無卡提款額度只有5,000

```
評分：
• 誠實告知額度限制 ✅ 沒有硬撐
• 提供替代方案（先玩，明天臨櫃轉帳）✅

⚠️ 需修正：
1. 「無法更您確認」→ 應為「無法跟您確認」
2. 「先賭禮」→ 應為「先繼續遊戲」或「先保留分數」，避免用「賭」字
3. 會員想回 3 萬但只能回 5,000 → 落差大，後續可能不滿，留意情緒
```

## 常見用詞修正

| 員工常用 | 建議修正 | 原因 |
|----------|----------|------|
| 贈禮/贈憶 | 儲值/入帳完成 | 用詞更精準 |
| 賭禮/先賭 | 繼續遊戲/保留分數 | 避免「賭」字 |
| 陪禮 | 儲值完成 | 字形錯誤 |
| 無法更您確認 | 無法跟您確認 | 錯字 |
