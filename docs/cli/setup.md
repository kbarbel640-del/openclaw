# `amigo setup`

Initialize `~/.amigo/amigo.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
amigo setup
amigo setup --workspace ~/.amigo/workspace
```

To run the wizard via setup:

```bash
amigo setup --wizard
```
