# `amigo tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
amigo tui
amigo tui --url ws://127.0.0.1:18789 --token <token>
amigo tui --session main --deliver
```
