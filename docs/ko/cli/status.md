# `openclaw status`

채널 + 세션에 대한 진단입니다.

```bash
openclaw status
openclaw status --all
openclaw status --deep
openclaw status --usage
```

참고:

- `--deep`는 실시간 프로브(WhatsApp Web + Telegram + Discord + Google Chat + Slack + Signal)를 실행합니다.
- 여러 에이전트가 구성된 경우, 출력에는 에이전트별 세션 스토어가 포함됩니다.
- 사용 가능한 경우, 개요에는 Gateway(게이트웨이) + 노드 호스트 서비스의 설치/런타임 상태가 포함됩니다.
- 개요에는 업데이트 채널 + git SHA(소스 체크아웃의 경우)가 포함됩니다.
- 업데이트 정보는 개요에 표시되며, 업데이트가 가능한 경우 상태는 `openclaw update` 실행을 안내하는 힌트를 출력합니다([Updating](/install/updating) 참조).
