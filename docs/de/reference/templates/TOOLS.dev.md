# TOOLS.md – Notizen zu Benutzerwerkzeugen (bearbeitbar)

Diese Datei ist für _Ihre_ Notizen zu externen Werkzeugen und Konventionen.
Sie definiert nicht, welche Werkzeuge existieren; OpenClaw stellt intern integrierte Werkzeuge bereit.

## Beispiele

### imsg

- Eine iMessage/SMS senden: beschreiben Sie, an wen/was, und bestätigen Sie vor dem Senden.
- Bevorzugen Sie kurze Nachrichten; vermeiden Sie das Senden von Geheimnissen.

### sag

- Text-zu-Sprache: geben Sie Stimme, Ziel‑Lautsprecher/Raum und an, ob gestreamt werden soll.

Fügen Sie alles hinzu, was der Assistent über Ihre lokale Toolchain wissen soll.
