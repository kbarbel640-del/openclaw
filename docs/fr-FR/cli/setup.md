# `openclaw setup`

Initialiser `~/.openclaw/openclaw.json` et l'espace de travail de l'agent.

Voir aussi :

- Premiers pas : [Premiers pas](/fr-FR/start/getting-started)
- Assistant : [Configuration initiale](/fr-FR/start/onboarding)

## Exemples

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

Pour exécuter l'assistant via setup :

```bash
openclaw setup --wizard
```
