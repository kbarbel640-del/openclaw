# `openclaw tui`

Ouvrir l'interface terminal connectée à la Passerelle.

Voir aussi :

- Guide TUI : [TUI](/fr-FR/web/tui)

## Exemples

```bash
openclaw tui
openclaw tui --url ws://127.0.0.1:18789 --token <token>
openclaw tui --session main --deliver
```
