# Recherche Mémoire Workspace

Système mémoire offline pour workspaces OpenClaw.

## Architecture

- **Store canonique** : Markdown files (git-friendly)
- **Index dérivé** : SQLite FTS5 (recall rapide)
- **Retain/Recall/Reflect** : boucle opérationnelle
