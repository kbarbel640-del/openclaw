# TOOLS.md - Användaranteckningar om verktyg (redigerbar)

Den här filen är till för _dina_ anteckningar om externa verktyg och konventioner.
Den definierar inte vilka verktyg som finns; OpenClaw tillhandahåller inbyggda verktyg internt.

## Exempel

### imsg

- Skicka ett iMessage/SMS: beskriv vem/vad, bekräfta innan du skickar.
- Föredra korta meddelanden; undvik att skicka hemligheter.

### sag

- Text-till-tal: ange röst, mål-högtalare/rum och om strömning ska användas.

Lägg till vad som helst annat som du vill att assistenten ska känna till om din lokala verktygskedja.
