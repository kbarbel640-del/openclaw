# TOOLS.md - 사용자 도구 참고 사항(편집 가능)

이 파일은 외부 도구 및 규칙에 대한 _사용자의_ 메모용입니다.
어떤 도구가 존재하는지 정의하지 않습니다. OpenClaw는 내부적으로 내장된 도구를 제공합니다.

## 예

### imsg

- iMessage/SMS 보내기: 누구/무엇을 설명하고 보내기 전에 확인하세요.
- 짧은 메시지를 선호합니다. 비밀을 보내지 마십시오.

### 처짐

- 텍스트 음성 변환: 음성, 대상 스피커/방, 스트리밍 여부를 지정합니다.

어시스턴트가 로컬 툴체인에 대해 알기를 원하는 내용을 추가하세요.
