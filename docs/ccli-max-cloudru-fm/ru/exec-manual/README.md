# Руководство пользователя: AI-ассистент в Telegram

**Для руководителей и топ-менеджеров**

Ваш персональный AI-ассистент доступен через Telegram-бота. Он понимает естественный язык, выполняет задачи и интегрирован с корпоративными сервисами Cloud.ru.

---

> **Важно:** На момент публикации (февраль 2026) интеграция с Telegram-каналами (`/tg_digest`, `/tg_channel`) временно недоступна из-за проблем с API-ключами. Функциональность будет добавлена в ближайшем обновлении.

---

## Быстрый старт

1. Откройте Telegram
2. Найдите бота (имя предоставит администратор)
3. Напишите любой вопрос или задачу

Бот ответит в течение нескольких секунд.

## Специальные команды

### Управление сессией

| Команда   | Описание                                |
| --------- | --------------------------------------- |
| `/new`    | Начать новый диалог (сбросить контекст) |
| `/status` | Проверить статус бота                   |

### Работа с AI Factory (облачные агенты)

| Команда                     | Описание                            |
| --------------------------- | ----------------------------------- |
| `/status_agents`            | Показать доступных облачных агентов |
| `/agents_on`                | Подключить облачных агентов         |
| `/agents_off`               | Отключить облачных агентов          |
| `/ask_agent <имя> <вопрос>` | Спросить конкретного агента         |

**Пример:**

```
/ask_agent weather-agent Какая погода в Москве?
```

### Почта и каналы

| Команда                        | Описание                                          |
| ------------------------------ | ------------------------------------------------- |
| `/email_brief [период]`        | Сводка Gmail (1d, 7d, 14d)                        |
| `/tg_digest [период]`          | Дайджест Telegram-каналов _(временно недоступно)_ |
| `/tg_channel <канал> [период]` | Сводка конкретного канала _(временно недоступно)_ |

## Работа с облачными агентами (AI Factory)

AI Factory — это платформа Cloud.ru с готовыми AI-агентами для специализированных задач.

### Расширение функциональности

Благодаря интеграции с **Evolution Cloud AI Factory** функциональность бота можно расширять по запросу:

- **Готовая библиотека AI-агентов** — специализированные агенты для различных бизнес-задач (аналитика, документооборот, HR, финансы и др.)
- **MCP-серверы** — дополнительные инструменты и интеграции, доступные через протокол Model Context Protocol
- **Кастомизация** — возможность подключения новых агентов под конкретные потребности бизнеса

Для подключения дополнительных агентов или MCP-серверов обратитесь к администратору системы.

### Как подключить

1. Отправьте `/status_agents` — увидите список доступных агентов
2. Отправьте `/agents_on` — агенты подключатся
3. Отправьте `/new` — начните новую сессию

### Как использовать

После подключения агенты доступны автоматически. Бот сам выберет нужного агента для вашей задачи.

Или обратитесь напрямую:

```
/ask_agent weather-agent Прогноз погоды на завтра
```

### Как отключить

```
/agents_off
/new
```

## Чего бот НЕ может

### Технические ограничения

- **Нет доступа к корпоративным системам** — бот не подключен к 1С, SAP, внутренним CRM
- **Нет доступа к файлам на вашем компьютере** — бот работает только в Telegram
- **Не может совершать звонки** — только текстовое общение
- **Не может бронировать** — отели, билеты, рестораны (только рекомендации)
- **Не может отправлять письма** — только читает почту, не отправляет

### Ограничения по данным

- **Знания ограничены датой обучения модели** — для актуальных данных используйте поиск
- **Не помнит предыдущие диалоги** — после `/new` контекст сбрасывается
- **Не имеет доступа к закрытым ресурсам** — только публичный интернет

### Что не стоит спрашивать

- Конфиденциальную информацию (пароли, ключи)
- Персональные данные третьих лиц
- Юридические или медицинские консультации (бот не заменяет специалистов)

## Техническая поддержка

Если бот не отвечает или работает некорректно:

1. Попробуйте `/new` — сброс сессии часто помогает
2. Проверьте `/status` — убедитесь, что бот активен
3. Подождите 30 секунд и повторите запрос

При системных проблемах обратитесь к администратору.
