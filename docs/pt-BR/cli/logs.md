# `openclaw logs`

Acompanhe logs de arquivos do Gateway via RPC (funciona em modo remoto).

Relacionado:

- Visao geral de logging: [Logging](/logging)

## Exemplos

```bash
openclaw logs

openclaw logs --json
openclaw logs --limit 500
```
