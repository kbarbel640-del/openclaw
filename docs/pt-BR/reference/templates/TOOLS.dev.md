# TOOLS.md - Notas de Ferramentas do Usuario (editavel)

Este arquivo e para as _suas_ notas sobre ferramentas externas e convencoes.
Ele nao define quais ferramentas existem; o OpenClaw fornece ferramentas integradas internamente.

## Exemplos

### imsg

- Enviar um iMessage/SMS: descreva quem/o que, confirme antes de enviar.
- Prefira mensagens curtas; evite enviar segredos.

### sag

- Texto para fala: especifique a voz, o alto-falante/sala de destino e se deve transmitir em streaming.

Adicione qualquer outra coisa que voce queira que o assistente saiba sobre sua cadeia de ferramentas local.
