# Theo dõi mức sử dụng

## Nó là gì

- Lấy mức sử dụng/hạn mức của nhà cung cấp trực tiếp từ các endpoint mức sử dụng của họ.
- Không ước tính chi phí; chỉ dùng các khung thời gian do nhà cung cấp báo cáo.

## Nó hiển thị ở đâu

- `/status` trong chat: thẻ trạng thái giàu emoji với token của phiên + chi phí ước tính (chỉ API key). Mức sử dụng của nhà cung cấp hiển thị cho **nhà cung cấp mô hình hiện tại** khi có sẵn.
- `/usage off|tokens|full` trong chat: chân trang mức sử dụng cho từng phản hồi (OAuth chỉ hiển thị token).
- `/usage cost` trong chat: tóm tắt chi phí cục bộ được tổng hợp từ log phiên OpenClaw.
- CLI: `openclaw status --usage` in ra bảng phân tích đầy đủ theo từng nhà cung cấp.
- CLI: `openclaw channels list` in ra cùng ảnh chụp mức sử dụng kèm theo cấu hình nhà cung cấp (dùng `--no-usage` để bỏ qua).
- Thanh menu macOS: mục “Usage” dưới Context (chỉ khi có sẵn).

## Nhà cung cấp + thông tin xác thực

- **Anthropic (Claude)**: token OAuth trong hồ sơ xác thực.
- **GitHub Copilot**: token OAuth trong hồ sơ xác thực.
- **Gemini CLI**: token OAuth trong hồ sơ xác thực.
- **Antigravity**: token OAuth trong hồ sơ xác thực.
- **OpenAI Codex**: token OAuth trong hồ sơ xác thực (dùng accountId khi có).
- **MiniMax**: API key (key gói coding; `MINIMAX_CODE_PLAN_KEY` hoặc `MINIMAX_API_KEY`); dùng khung thời gian gói coding 5 giờ.
- **z.ai**: API key qua env/config/kho xác thực.

Mức sử dụng sẽ bị ẩn nếu không có thông tin xác thực OAuth/API tương ứng.
