# スクリプト

`scripts/` ディレクトリには、ローカルのワークフローや運用タスク向けの補助スクリプトが含まれています。
タスクが明確にスクリプトに結び付いている場合に使用し、それ以外の場合は CLI を優先してください。

## 規約

- スクリプトは、ドキュメントやリリースチェックリストで参照されていない限り **任意** です。
- 既存の CLI インターフェースがある場合は、そちらを優先してください（例: 認証監視は `openclaw models status --check` を使用します）。
- スクリプトはホスト固有であると想定し、新しいマシンで実行する前に内容を確認してください。

## スクリプトを追加する場合

- スクリプトは目的を明確にし、ドキュメント化してください。
- 関連するドキュメントに短いエントリを追加してください（存在しない場合は作成してください）。
