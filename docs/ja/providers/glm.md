# GLM モデル

GLM は Z.AI プラットフォームを通じて提供される **モデルファミリー**（企業ではありません）です。OpenClaw では、GLM モデルは `zai` プロバイダーと、`zai/glm-4.7` のようなモデル ID を介して利用します。

## CLI セットアップ

```bash
openclaw onboard --auth-choice zai-api-key
```

## 設定スニペット

```json5
{
  env: { ZAI_API_KEY: "sk-..." },
  agents: { defaults: { model: { primary: "zai/glm-4.7" } } },
}
```

## 注意事項

- GLM のバージョンや提供状況は変更される可能性があります。最新情報については Z.AI のドキュメントを確認してください。
- モデル ID の例には `glm-4.7` や `glm-4.6` があります。
- プロバイダーの詳細については、[/providers/zai](/providers/zai) を参照してください。
