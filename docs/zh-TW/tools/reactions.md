# 反應工具

跨頻道共用的反應語意：

- 新增反應時需要 `emoji`。
- `emoji=""` 在支援時會移除機器人的反應。
- `remove: true` 在支援時會移除指定的表情符號（需要 `emoji`）。

頻道注意事項：

- **Discord/Slack**：空的 `emoji` 會移除訊息上機器人的所有反應；`remove: true` 僅移除該表情符號。
- **Google Chat**：空的 `emoji` 會移除訊息上應用程式的反應；`remove: true` 僅移除該表情符號。
- **Telegram**：空的 `emoji` 會移除機器人的反應；`remove: true` 也會移除反應，但工具驗證仍需要非空的 `emoji`。
- **WhatsApp**：空的 `emoji` 會移除機器人反應；`remove: true` 會對應為空表情符號（仍需要 `emoji`）。
- **Signal**：當啟用 `channels.signal.reactionNotifications` 時，傳入的反應通知會發出系統事件。
