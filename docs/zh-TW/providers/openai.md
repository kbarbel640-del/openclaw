# OpenAI

OpenAI 提供適用於 GPT 模型的開發者 API。Codex 支援 **ChatGPT 登入** 以使用訂閱存取，或 **API 金鑰** 登入以進行依用量計費的存取。Codex 雲端需要 ChatGPT 登入。

## 選項 A：OpenAI API 金鑰（OpenAI Platform）

**最適合：** 直接 API 存取與依用量計費。  
請從 OpenAI 儀表板取得你的 API 金鑰。

### CLI 設定

```bash
openclaw onboard --auth-choice openai-api-key
# or non-interactive
openclaw onboard --openai-api-key "$OPENAI_API_KEY"
```

### 設定片段

```json5
{
  env: { OPENAI_API_KEY: "sk-..." },
  agents: { defaults: { model: { primary: "openai/gpt-5.1-codex" } } },
}
```

## 選項 B：OpenAI Code（Codex）訂閱

**最適合：** 使用 ChatGPT／Codex 訂閱存取，而非 API 金鑰。  
Codex 雲端需要 ChatGPT 登入，而 Codex CLI 支援 ChatGPT 或 API 金鑰登入。

### CLI 設定

```bash
# Run Codex OAuth in the wizard
openclaw onboard --auth-choice openai-codex

# Or run OAuth directly
openclaw models auth login --provider openai-codex
```

### 設定片段

```json5
{
  agents: { defaults: { model: { primary: "openai-codex/gpt-5.3-codex" } } },
}
```

## 注意事項

- 模型參照一律使用 `provider/model`（請見 [/concepts/models](/concepts/models)）。
- 驗證細節與重用規則請見 [/concepts/oauth](/concepts/oauth)。
