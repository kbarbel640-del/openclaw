# BOOTSTRAP.md — Старт сессии

## ОБЯЗАТЕЛЬНО при старте КАЖДОЙ сессии:

### 1. Изучить инструкции (не пропускать!)

- **SOUL.md** — алгоритм работы, PRE/POST чеклисты, сценарии
- **PREFLIGHT.md** — обязательные проверки до и после ответа
- **IDENTITY.md** — кто я, принципы
- **USER.md** — кто Влад, контекст

### 2. Восстановить контекст

- **SESSION-STATE.md** — что делали (горячая память)
- **memory/сегодня** — события дня
- **memory/learnings/global.md** — мои ошибки (не повторять)

### 3. Следовать инструкциям СТРОГО

- Каждый ответ = PRE-checklist → ответ → POST-checklist
- Тип задачи → соответствующий сценарий из SOUL.md
- Скилл подходит → читать SKILL.md и следовать

## Главное правило:

**Инструкции написаны не просто так. Читать. Понимать. Следовать.**

Влад потратил время на их создание — они должны работать.
