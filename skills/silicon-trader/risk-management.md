# 风险管理规则

> 铁律，不可违背

---

## 仓位管理

### 单笔风险
```
标准: 账户净值 × 1%
最大: 账户净值 × 2% (仅特殊情况)
```

### 仓位计算公式
```
风险金额 = 账户净值 × 风险比例
手数 = 风险金额 ÷ (止损点数 × 每点价值)
```

### 计算示例
```
账户净值: $10,000
单笔风险: 1% = $100
止损距离: 60 pips
XAUUSD点值: $1/pip per 0.01 lot → $10/pip per 0.1 lot

手数 = $100 ÷ (60 × $10) = 0.167 lot
取整: 0.16 lot (向下取整)
```

## 日内限制

### 最大日亏损
| 阈值 | 动作 |
|------|------|
| -3% | **强制停机**，当日不再交易 |
| -5% | 次日也休息，复盘后恢复 |

### 连续亏损
| 连亏次数 | 动作 |
|----------|------|
| 3次 | 暂停1小时 |
| 5次 | 当日停机 |
| 7次 | 停机3天 + 策略复盘 |

## 止损规则

### 初始止损
- 放置在扫荡点（Sweep High/Low）之外 **10-15 pips**
- 必须在开仓时设置
- **禁止无止损交易**

### 移动止损
| 条件 | 动作 |
|------|------|
| TP1触及 | 止损移至保本 (Entry) |
| TP1.5触及 | 可选追踪止损 |

### 禁止行为
- ❌ 不设止损
- ❌ 扩大止损（"再给它一点空间"）
- ❌ 亏损时加仓（摊平成本）
- ❌ 扛单过夜（除非计划内）

## 新闻回避

### 高影响事件
以下事件前后 **30分钟** 禁止开新仓：

| 事件 | 全称 | 影响 |
|------|------|------|
| CPI | Consumer Price Index | 通胀数据 |
| NFP | Non-Farm Payrolls | 就业数据 |
| FOMC | Federal Open Market Committee | 利率决议 |
| GDP | Gross Domestic Product | 经济增长 |
| PPI | Producer Price Index | 生产者物价 |

### 新闻处理
| 状态 | 处理 |
|------|------|
| 无持仓 | 等数据公布+市场稳定后再入场 |
| 有持仓盈利 | 考虑提前平仓锁定利润 |
| 有持仓亏损 | 收紧止损或提前止损 |

## 点差控制

### XAUUSD 点差限制
| 点差范围 | 状态 | 动作 |
|----------|------|------|
| < 30 pips | 🟢 正常 | 可以交易 |
| 30-50 pips | 🟡 警戒 | 谨慎交易 |
| > 50 pips | 🔴 禁止 | 停止所有挂单 |

### 检查时机
- 每次下单前检查实时点差
- 高波动时段特别注意（新闻、开盘）
- 亚洲盘流动性差时点差可能扩大

## 每日检查清单

### 📋 开盘前
```
□ 检查账户净值
□ 计算今日最大可亏损金额 (-3%)
□ 查看当日重要新闻时间
□ 确认交易时段 (伦敦15:30/纽约20:30)
□ 检查昨日复盘是否完成
```

### 📋 交易中
```
□ 每笔交易计算正确仓位
□ 确认止损已设置
□ 监控累计盈亏
□ 避开新闻时段
```

### 📋 收盘后
```
□ 记录当日所有交易
□ 更新复盘日志
□ 评估是否触发熔断
□ 计划明日关注点
```

## 熔断机制

### 触发条件
| 条件 | 熔断级别 |
|------|----------|
| 日亏损 ≥ 3% | Level 1 - 当日停机 |
| 连续亏损 ≥ 5笔 | Level 1 - 当日停机 |
| 日亏损 ≥ 5% | Level 2 - 停机2天 |
| 周亏损 ≥ 10% | Level 3 - 停机1周 + 策略审查 |

### 熔断后流程
```
1. 立即停止所有交易
2. 平仓所有持仓（可选）
3. 记录触发原因
4. 分析问题根源
5. 制定改进方案
6. 等待冷静期结束
7. 复盘确认后恢复
```

### 恢复条件
| 熔断级别 | 恢复条件 |
|----------|----------|
| Level 1 | 次日自动恢复 |
| Level 2 | 完成复盘 + 手动确认 |
| Level 3 | 策略审查 + 模拟盘验证 |

## 心理规则

### ✅ 应该做
- 接受亏损是交易的一部分
- 严格执行计划
- 保持交易日志
- 定期复盘

### ❌ 不应该做
- 报复性交易（亏损后加大仓位）
- FOMO（害怕错过）
- 改变计划（临时调整止损/止盈）
- 过度交易（一天交易太多次）

## 资金保护公式

### 凯利公式（参考）
```
f = (bp - q) / b

f = 最优仓位比例
b = 盈亏比
p = 胜率
q = 1 - p = 败率
```

### 保守建议
- 无论凯利公式结果如何，单笔不超过2%
- 新策略/新市场：先用0.5%测试
- 连续盈利后不放大仓位
