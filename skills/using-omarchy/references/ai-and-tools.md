## AI in Omarchy

- Default agent harness: OpenCode (launch via `c` alias in a repo).
- Dictation: install via Omarchy menu Install > AI > Dictation; trigger with Super + Ctrl + X.
- Local LLMs: LM Studio and Ollama are recommended.
- Other harnesses (install via Omarchy menu): Claude Code, Gemini CLI, OpenAI Codex, GitHub Copilot.
- Omarchy skill is bundled for agent assistance; run in plan mode first and be ready to roll back.

## Terminal

- Default terminal: Ghostty (tabs + splits).
- Alternate supported terminals: Alacritty, Kitty (install via Omarchy menu > Install > Terminal).
