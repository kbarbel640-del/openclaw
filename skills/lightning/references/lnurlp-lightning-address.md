# LNURLp + Lightning Address (LNbits)

## LNURLp (LNbits extension)

- LNbits LNURLp extension lets you create LNURL-pay links and attach a username to create a Lightning Address.
  Source:
- https://github.com/lnbits/lnurlp

## Lightning Address (LNbits guidance)

- LNbits docs describe Lightning Address as username@domain with the LNURLp extension.
  Source:
- https://docs.lnbits.org/guide/lnurlp.html
