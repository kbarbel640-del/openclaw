# UX Specs (v0)

This directory contains operator-facing UX specifications for the v0 dispatch release.

## Artifacts
- `dispatcher_cockpit_v0.md` - dispatcher queue, SLA controls, assignment override, timeline view.
- `technician_job_packet_v0.md` - field packet, execution checklist, evidence/closeout requirements.

## Scope
These are specification artifacts (not implemented UI code).
They are intended to drive downstream design and frontend implementation work.
