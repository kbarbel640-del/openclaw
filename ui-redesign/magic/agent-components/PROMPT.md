# Agent Status Components

## Source
Magic MCP Server - 21st_magic_component_builder

## Search Query
"agent status card"

## Message
Build an AgentStatusCard component that shows an AI agent's current status (active, idle, busy, error), current task if running, last activity time, and action buttons. Should have expanded and compact variants.

## Standalone Request Query
Create an AgentStatusCard component with status indicators (active with pulsing dot, idle, busy, error states), current task display, last activity timestamp, and action buttons (pause/resume, configure). Include expanded and compact variants for use in lists and detail views.

## Component Generated
**AgentStatusCard** - Full-featured agent status card with:
- Status indicators (active with pulsing animation, idle, busy with spinner, error)
- HyperText animation for "PROCESSING" label
- Current task display
- Last activity timestamp
- Pause/Resume and Configure action buttons
- Expanded and Compact variants
- Framer Motion animations
